---
title: Create your first Thread
---

This guide assumes that you have set up authentication, if not, checkout [authentication guide](/get-started/authentication).

<CodeGroup>
```python Python
import os
from literalai import LiteralClient

client = LiteralClient(api_key=os.getenv("LITERAL_API_KEY"))

@client.step(type="run")
def my_step(input):
  return f"World"

@client.thread
def main():
  print(my_step("Hello"))

main()
client.wait_until_queue_empty()
print("Done")

```
</CodeGroup>

Run the code, you should see something similar to the following output:

<CodeGroup>
```bash python
$ python step-example.py
World
Done
````
</CodeGroup>
