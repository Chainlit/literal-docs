title: Create your first Thread
---

This guide assumes that you have set up authentication, if not, checkout [authentication guide](/get-started/authentication).

<CodeGroup>
```python first-thread.py
import os
from literalai import LiteralClient

client = LiteralClient(api_key=os.getenv("LITERAL_API_KEY"))

@client.step(type="run")
def my_step(input):
  return f"World"

@client.thread
def main():
  print(my_step("Hello"))

main()
client.wait_until_queue_empty()
print("Done")
```


```typescript first-thread.ts
import { LiteralClient } from '@literalai/client';

const client = new LiteralClient(process.env['LITERAL_API_KEY']);

const thread = await client.thread().upsert();

const step = await thread
  .step({ type: "run", name: "firstStep" })
  .send();
```
</CodeGroup>

Run the code, you should see something similar to the following output:

<CodeGroup>
```bash python
python first-thread.py
```
```typescript TypeScript
npx ts-node first-thread.ts
```
</CodeGroup>
