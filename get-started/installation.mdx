## Instantiate the Literal AI client

<CodeGroup>
```python Python
import os
from literalai import LiteralClient

# Not compatible with gunicorn's --preload flag
literal_client = LiteralClient(api_key=os.getenv("LITERAL_API_KEY")) #Â This is the default and can be omitted
```

```typescript TypeScript
import { LiteralClient } from '@literalai/client';

const literalClient = new LiteralClient(process.env['LITERAL_API_KEY']); // This is the default and can be omitted
```
</CodeGroup>

## How to get my API key?

To get your API key, go to your [project page](https://cloud.getliteral.ai) and click on the **Settings** tab. You will find your API key in the **API Key** section.

## Log your first thread

<CodeGroup>
```python Python
import os
from literalai import LiteralClient

literal_client = LiteralClient(api_key=os.getenv("LITERAL_API_KEY"))

with literal_client.thread(name="My first thread !") as thread:
  literal_client.message(content="Hello world", type="user_message", name="My first step !")

literal_client.flush_and_stop()
```

```typescript TypeScript
import { LiteralClient } from '@literalai/client';

const literalClient = new LiteralClient(process.env['LITERAL_API_KEY']);

const thread = await literalClient
  .thread({ name: "My first thread !" })
  .upsert();

await thread
  .step({
    type: "user_message",
    name: "My first step !",
    output: { content: "Hello world" },
  })
  .send();
```
</CodeGroup>

You can now go to your [project page](https://cloud.getliteral.ai) and see your first thread.
