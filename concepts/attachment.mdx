## Create an attachment:

<CodeGroup>
```python Python
attachment = await client.api.create_attachment(
    thread_id="<THREAD_UUID>",
    step_id="<STEP_UUID>"
    name="my_attachment",
    content_type="text/plain",
    content="Hello world",
)
```
</CodeGroup>
