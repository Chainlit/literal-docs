A user session can be created to track interactions with you LLM app.
User sessions are then used to compute metrics on the dashboard.

<Frame caption="Literal analytics page">
  <img src="/images/user-analytics.png" />
</Frame>

You can simply create a session with:

<CodeGroup>

```python Python
from literalai import LiteralClient

client = LiteralClient()

await client.api.create_user_session(is_interactive=True, participant_identifier="alex@example.com")

```

</CodeGroup>

## Advanced usage

### Anonymous User

If you don't have an authentication mechanism for your users, you can pass a unique identifier as anon_participant_identifier. This unique identifier can be an IP address, a user agent, or any data that combines multiple user characteristics.

```python Python
from literalai import LiteralClient

client = LiteralClient()
def create_user_session(request)
  id = hash(request.ip + request.user_agent)
  await client.api.create_user_session(is_interactive=True, anon_participant_identifier=id)
```
