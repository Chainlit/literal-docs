```python

    thread_id: "1234"
    type = "undefined",
)
```

## Send Steps

Used to send multiple steps at the same time to the Chainlit platform.

```python
steps = [
    {
        "thread_id": "1234",
        "type": "undefined",
    },
    {
        "thread_id": "1234",
        "type": "undefined",
    },
]

client.api.send_steps(steps=[Step.from_dict(step) for step in steps])
```
