## Create a feedback

```python
feedback = client.api.create_feedback(
    thread_id="1234",
    value=1,
    comment="Hello world",
)
```

<ParamField path="step_id" type="string">
  Step id in string format
</ParamField>

<ParamField path="value" type="int">
  Value in integer format, 1 for positive feedback and 0 for negative feedback
</ParamField>

<ParamField path="comment" type="str">
  Comment in string format
</ParamField>

<ResponseField name="strategy" type="string">
  <Expandable title="properties">
    <ResponseField name="BINARY"></ResponseField>
    <ResponseField name="STARS"></ResponseField>
    <ResponseField name="BIG_STARS"></ResponseField>
    <ResponseField name="LIKERT"></ResponseField>
    <ResponseField name="CONTINUOUS"></ResponseField>
    <ResponseField name="LETTERS"></ResponseField>
    <ResponseField name="PERCENTAGE"></ResponseField>
  </Expandable>
</ResponseField>

## Update a feedback

```python

feedback = client.api.update_feedback(
    id="1234",
    update_params={
        "comment": "Hello world",
    },
)
```

<ParamField path="id" type="string">
  Id in string format
</ParamField>

<ParamField path="update_params" type="dict">
  Dict containing same parameters as create_feedback
</ParamField>
