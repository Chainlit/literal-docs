## Create an User

```python
from chainlit_client import ChainlitClient

client = ChainlitClient()
new_user = client.api.create_user(identifier="John Doe")
```

<ParamField path="identifier" type="string" required>
  The identifier of the user
</ParamField>

### Return type

<ResponseField name="user" type="User">
  Return the user
</ResponseField>

## Get an User

```python
user = client.api.get_user(id="adc52cca-44ef-49b5-9c6b-af9d3b8c5ee4")
```

<ParamField path="id" type="uuid" required>
  The id of the user
</ParamField>

### Return type

<ResponseField name="user" type="User">
  Return the user, can be None if the user does not exist
</ResponseField>

## Update an User

```python
updated_user = client.api.update_user(id="c0bd9bf3-45f2-447b-ae67-d4c18ae06542", identifier="Jane Doe")
```
