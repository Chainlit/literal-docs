This is a short toturial to get you started with the Chainlit Python client.

## Installation

```bash

pip install chainlit-sdk dotenv openai asyncio

```

## Setup

Create a .env file in the root of your project and add the following:

```text
OPENAI_API_KEY="<your-openai-api-key>"
CHAINLIT_API_KEY="<your-chainlit-api-key>"
```

## Usage

Copy the following code into a file called `main.py` and run it.

```python
import asyncio
import json

from chainlit_client import ChainlitClient
from dotenv import load_dotenv
from openai import OpenAI

load_dotenv()

openai_client = OpenAI()

client = ChainlitClient()
client.instrument_openai()


@client.thread
def run():
    welcome_message = "What's your name? "
    client.message(content=welcome_message, type="system_message")

    text = input(welcome_message)
    client.message(content=text, type="user_message")

    completion = openai_client.chat.completions.create(
        model="gpt-3.5-turbo",
        messages=[
            {
                "role": "system",
                "content": "Tell an inspiring quote to the user, mentioning their name. Be supportive while keeping it short. Write one sentence per line.",
            },
            {
                "role": "assistant",
                "content": welcome_message,
            },
            {
                "role": "user",
                "content": text,
            },
        ],
    )
    print(completion.choices[0].message.content)
    client.message(content=completion, type="assistant_message")


run()
client.wait_until_queue_empty()
```
